# <p align="center">Групповой проект YaMDb студентов Яндекс.Практикум команды №59</p>

Проект YaMDb собирает отзывы пользователей на произведения.

**Сами произведения в YaMDb не хранятся, здесь нельзя посмотреть фильм или послушать музыку.**

Произведения делятся на категории, такие как «Книги», «Фильмы», «Музыка». Например, в категории «Книги» могут быть произведения «Винни-Пух и все-все-все» и «Марсианские хроники», а в категории «Музыка» — песня «Давеча» группы «Жуки» и вторая сюита Баха. Список категорий может быть расширен (например, можно добавить категорию «Изобразительное искусство» или «Ювелирка»). 

Произведению может быть присвоен жанр из списка предустановленных (например, «Сказка», «Рок» или «Артхаус»). 
Добавлять произведения, категории и жанры может только администратор.

Благодарные или возмущённые пользователи оставляют к произведениям текстовые отзывы и ставят произведению оценку в диапазоне от одного до десяти (целое число); из пользовательских оценок формируется усреднённая оценка произведения — рейтинг (целое число). На одно произведение пользователь может оставить только один отзыв.
Пользователи могут оставлять комментарии к отзывам.

*Добавлять отзывы, комментарии и ставить оценки могут только аутентифицированные пользователи.*


## Возможности

- Ресурс **auth**: аутентификация.
- Ресурс **users**: пользователи.
- Ресурс **titles**: произведения, к которым пишут отзывы (определённый фильм, книга или песенка).
- Ресурс **categories**: категории (типы) произведений («Фильмы», «Книги», «Музыка»). Одно произведение может быть привязано только к одной категории.
- Ресурс **genre**: жанры произведений. Одно произведение может быть привязано к нескольким жанрам.
- Ресурс **reviews**: отзывы на произведения. Отзыв привязан к определённому произведению.
- Ресурс **comments**: комментарии к отзывам. Комментарий привязан к определённому отзыву.

## Установка
Используйте python3 для MacOS и Linux

1. Клонируйте репозиторий:

    ```python
    git clone git@github.com:VPanshin/Yandex-Practicum-Study-Projects.git
    ```

2. Cоздайте и активировайте виртуальное окружение:

    ```python
    $ cd Yandex-Practicum-Study-Projects/5_API_for_YamDB
    $ python -m venv venv

    $ source venv/Scripts/activate  #Для Windows
    $ source venv/bin/activate #Для MacOs/Linux
   ```

3. Установите зависимости:

    ```py
    (venv) $ python -m pip install --upgrade pip
    (venv) $ pip install -r requirements.txt
    ```

4. Примените миграции базы данных:

    ```py 
    cd api_yamdb/
    python manage.py migrate
    ```

5. Запустите проект:

    ```bash
    python manage.py runserver
    ```

## Примеры использования

**Некоторые примеры запросов к API:**

- Регистрация нового пользователя [POST]:

    `api/v1/auth/signup/`

- Получение данных своей учетной записи [GET]:

    `/api/v1/users/me/`

- Получение списка всех отзывов [GET]:

    `/api/v1/titles/{title_id}/reviews/`

Полный перечен эндпоинтов доступен из документации к API по [адресу](http://127.0.0.1:8000/redoc/) после запуска сервера с проектом.
